##### Build Setup

``` bash
# clone repo
git clone http://192.168.10.158/InfcnFrontDev/qingdao_hotspot.git

# install dependencies
npm install

# bower install dependencies
bower install

# clean for production
npm run clean

# run for production
npm run dev

# build for production
npm run build
```
